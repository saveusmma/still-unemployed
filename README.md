# Still Unemployed?

A daily check-in app. The couch is waiting.

_(your Vercel URL goes here once deployed)_

## What's in this folder
- `src/App.jsx` — the actual app
- `src/main.jsx` — React entry point
- `index.html` — page wrapper
- `public/` — icons, manifest
- `package.json` — dependencies
- `vite.config.js` — build config

## Running locally (optional, for later)
You don't need this to deploy — Vercel handles building. But if you ever want
to run the app on your own computer:
```
npm install
npm run dev
```

## Deploying
This project is set up to deploy automatically on Vercel. Push to GitHub,
connect the repo on vercel.com, and Vercel handles the rest.
